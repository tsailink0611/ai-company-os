## 各ポジションの権限範囲

### オーナー（人間）
- 単独決定：新規事業の立ち上げ・撤退 / 月額10万超の発注・契約
- エスカレーション受信：CEO AIから判断依頼が来た時のみ対応
- 日常業務：CEO AIに指示を出すだけ

### CEO AIエージェント
- 単独決定：統括マネージャーへの委任 / ファイナルチェック合否判断
- オーナーへ確認：エスカレーション条件に該当する事項
- 禁止：統括マネージャーを経由せず部門へ直接指示する

### 経営管理部（CEO直轄）
- 自律実行：週次ビジネス健全性レポート
- CEOへ報告：財務・稼働・攻守ポジション判定
- 禁止：freee-mcpの書き込み操作の自律実行 / CEOを経由せずオーナーへ直接連絡

### 統括マネージャー
- 単独決定：部門への作業指示 / 品質チェック合否 / 差し戻し（最大2回）
- CEOへ報告：承認済み成果物 / エスカレーション相当の問題
- 禁止：CEOを経由せずオーナーへ直接連絡する

### 各部門エージェント
- 単独実行：統括マネージャーからの指示に基づく成果物作成
- 統括マネージャーへ報告：完了報告・不明点の確認
- 禁止：CEO・オーナーへの直接連絡

## 品質チェックフロー

```
部門 → 統括マネージャー（1次チェック）
  不合格 → 部門へ差し戻し（1回目）
  不合格 → 部門へ差し戻し（2回目）
  2回後も不合格 → CEO AIへエスカレーション
  合格 → CEO AIへ承認報告

CEO AI（ファイナルチェック）
  不合格 → 統括マネージャーへ差し戻し（最大2回）
  合格 → オーナーへ完了報告
```

## エスカレーション条件一覧

| 条件 | 担当 |
|------|------|
| 月額10万円超の発注・契約・課金 | オーナー |
| 実在する顧客への初回メール送信の実行 | オーナー |
| 新しいサービス価格・プラン内容の正式決定 | オーナー |
| 本番データの書き込み・削除 | オーナー |
| 新規事業の立ち上げ・撤退判断 | オーナー |
| 成果物が2回差し戻し後も改善されない | オーナー |
| 未収金が月商の30%超 | オーナー（経営管理部経由） |
| 稼働案件が上限超過で新規依頼あり | オーナー（経営管理部経由） |
